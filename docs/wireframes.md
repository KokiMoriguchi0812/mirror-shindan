# 画面設計（ワイヤーフレーム）

---

## 画面一覧

| 画面名 | URL | 説明 |
|--------|-----|------|
| トップページ | `/` | 診断の説明・開始ボタン |
| 診断ページ | `/quiz` | 32問の質問フォーム |
| 結果ページ | `/result/[token]` | 自己像・他者像・ギャップ表示 |
| 友人診断ページ | `/friend/[token]` | 友人が回答するページ |

## 2. 診断ページ `/quiz`

```
┌─────────────────────────────────────┐
│  STEP 2 / 4   行動様式 対人距離 …   │  ← 軸名クリックでジャンプ可
│  ████████████░░░░░░░░░░░░           │  ← プログレスバー
│                                     │
│  🌿 対人距離                        │  ← 軸タイトル
│  人との関わり方・社交性              │
│                                     │
│  ┌─────────────────────────────┐   │
│  │ Q1                          │   │
│  │  初めて会う人に、自分から    │   │
│  │  積極的に話しかけられる      │   │
│  │  ⬤  ◉  ·  ◉  ⬤            │   │
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │ Q2  … （8問分縦に並ぶ）      │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌──────────┐   ┌──────────────┐   │
│  │  ← 前へ  │   │  次へ →      │   │  ← 未回答でも移動可
│  └──────────┘   └──────────────┘   │
│                                     │
└─────────────────────────────────────┘
```

**スケールUI仕様：**
- 5つの●を横並びで表示
- 最左：大きい・濃い青（いいえ）
- 中央：小さい・薄い青（ラベルなし）
- 最右：大きい・濃い青（はい）
- 選択した●はアニメーションで強調

**ナビゲーション仕様：**
- 前へ：前の軸に戻る（最初の軸では非表示）
- 次へ：次の軸に進む（未回答でも可）
- 軸名クリック：任意の軸にジャンプ
- 最終軸では「診断結果を見る」ボタン（全32問回答済みのみ活性）
- 各軸の問題はページロード時にシャッフル
