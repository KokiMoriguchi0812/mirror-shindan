# 画面設計（ワイヤーフレーム）

---

## 画面一覧

| 画面名 | URL | 説明 |
|--------|-----|------|
| トップページ | `/` | 診断の説明・開始ボタン |
| 診断ページ | `/quiz` | 32問の質問フォーム |
| 結果ページ | `/result/[token]` | 自己像・他者像・ギャップ表示 |
| 友人診断ページ | `/friend/[token]` | 友人が回答するページ |

---

## 2. 診断ページ `/quiz`

```
┌─────────────────────────────────────┐
│  ████████████████░░░░░░  Q16 / 32   │  ← プログレスバー
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │  友人に急に誘われても、      │   │
│  │  その場でほぼ即決できる      │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
│         ⬤    ◉    ·    ◉    ⬤      │
│       いいえ               はい     │
│                                     │
│  ┌──────────┐   ┌──────────────┐   │
│  │  ← 前へ  │   │  次へ →      │   │
│  └──────────┘   └──────────────┘   │
│                                     │
└─────────────────────────────────────┘
```

**スケールUI仕様：**
- 5つの●を横並びで表示
- 最左：大きい・濃い青（いいえ）
- 中央：小さい・薄い青（ラベルなし）
- 最右：大きい・濃い青（はい）
- 選択した●はアニメーションで強調
