# 拡張機能リスト

今後のアップデートで追加を検討する機能の候補。

---

## フェーズ2（優先度：高）

- [ ] **メールアドレス任意登録**：結果URLをメールで受け取れる・友人回答時に通知
- [ ] **再診断機能**：過去の結果を保持したまま新しく診断し直せる
- [ ] **16タイプ詳細ページ**：各タイプの詳細解説・有名キャラクターとの対応
- [ ] **相性マトリクス**：16×16の相性一覧表（一目で相性確認）
- [ ] **相性詳細ページ**：2タイプを選ぶとなぜ合う/合わないか・付き合い方を解説

## フェーズ3（優先度：中）

- [ ] **ストレス時タイプ表示**：通常タイプとストレス時タイプを分けて表示
- [ ] **複数人比較ビュー**：友人A・友人B・自己像を並べて比較できる画面
- [ ] **タイプ別コミュニティ**：同じタイプ同士が交流できる掲示板
- [ ] **診断履歴**：過去の診断結果を時系列で確認（成長・変化の可視化）
- [ ] **シェア画像生成**：OGP画像・結果カード画像を自動生成（SNS投稿用）

## フェーズ4（マネタイズ強化）

- [ ] **詳細レポートPDF**：有料（Stripe決済）で詳細な性格分析PDFを生成
- [ ] **企業向けチーム診断**：チームメンバー全員の診断結果をダッシュボードで管理
- [ ] **Google AdSense 導入**：無料ユーザー向け広告表示
- [ ] **アフィリエイト連携**：タイプ別のおすすめ書籍・サービスの紹介

## フェーズ5（コンテンツ展開）

- [ ] **noteアカウント連携**：MBTIとは別の専用noteアカウントでタイプ別コンテンツ配信
- [ ] **グッズ展開**：SUZURIでタイプ別グッズ販売
- [ ] **英語対応**：海外展開（ただし日本文化特化の観点から慎重に検討）
- [ ] **アプリ化**：PWA対応またはネイティブアプリ

## アイデアメモ

- 風林火山タイプへの特別な演出（専用アニメーション・称号）
- 診断結果の壁紙生成機能
- パートナーと診断結果を比較する「カップル診断モード」
- 誕生日・星座との相関分析
