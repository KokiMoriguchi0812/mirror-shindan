# 要件定義書

## 1. システム概要

「自己像 vs 他者像のギャップ可視化」をコアとした日本人向け性格診断Webアプリ。
ログイン任意で診断でき、友人に専用URLを送ることで他者からの評価も取得できる。
アカウント登録することで診断結果を保存・管理できる。

---

## 2. 機能要件

### 2-1. 自己診断機能
- 4軸×8問（計32問）を軸ごとに1セットとして表示する
- 各軸8問を一画面に縦リスト表示し、「次へ」ボタンで次軸へ進む（未回答でも移動可）
- 「前へ」ボタンで前の軸に戻れる
- 上部の軸名（行動様式・対人距離・感情表現・価値基準）をクリックすると任意の軸にジャンプできる
- 各軸の問題はページロード時にランダムシャッフルされる
- 診断フロー：行動様式→対人距離→感情表現→価値基準→診断結果
- 全32問に回答後に送信可能（送信ボタンは未回答がある間は非活性）
- 回答完了後、16タイプのうち1つを判定する
- 判定結果を永続的に保存する
- 結果ページの専用URL（結果トークン）を発行する
- 友人診断用の別URL（友人トークン）を発行する

### 2-2. 友人診断機能
- 友人トークンURLを開くと「依頼者について答えてください」と表示する
- 自己診断と同じUI（前へ・次へ・軸クリック・シャッフル）で32問を表示する
- 複数人が回答可能（人数上限なし）
- 回答後にタブ形式で以下を表示する：
  - あなたから見た友人（タイプカード＋説明文）
  - 友人から見た友人・本人の自己診断（タイプカード＋説明文）
  - あなたから見た友人 VS 友人から見た友人のギャップ（友人向けメッセージ）

### 2-3. ギャップ比較・結果表示機能
- タブ形式で折りたためるセクション（デフォルト：全開）
- セクション順：自己像 → タイプ特徴 → 他者像（他人から見る自分）→ 自己像VS他者像のギャップ
- 各タイプのコード名・キャラ名・詳細説明文を表示する
- スコアゲージ：上部に特徴テキスト（即断即行/熟考慎重など）、左右に「風(F)」「岩(G)」形式で常時表示
- 友人回答数を「他者像（他人から見る自分）(N人の平均)」として表示

### 2-4. シェア機能
- 結果URLをコピー・LINE・Xでシェアするボタン
- 友人診断URLをコピー・LINEシェアするボタン

### 2-5. 認証・アカウント機能
- メールアドレス＋パスワードでアカウント登録（メール確認あり）
- ログイン・ログアウト機能
- パスワードリセット（メールリンク方式）
- ログイン任意：ゲストでも診断可能
- ログイン済みは診断完了時点で自動保存
- 未ログインの場合、結果ページに「ログインして保存」ボタンを表示
- マイページ：過去の診断履歴（コード名・キャラ名・画像・診断日）を一覧表示
- 結果ページ下部に再診断ボタン・過去の診断履歴を表示（ログイン時のみ）

### 2-6. トップページ
- 16タイプスライダー（左右矢印でタップ移動、インジケーター付き）
- 使い方3ステップ表示
- ヘッダーにログイン・登録ボタン（ログイン後：マイページ・ログアウト）

---

## 3. 非機能要件

### 3-1. パフォーマンス
- トップページの初回表示：3秒以内
- 診断結果の表示：2秒以内
- モバイルファーストで設計する

### 3-2. セキュリティ
- 結果URLは推測困難なUUIDベースのトークンを使用
- 友人診断URLは結果URLとは別トークンを使用
- Supabase Row Level Security (RLS) を設定する
- 認証はSupabase Authに委譲（メール確認・パスワードハッシュ化）

### 3-3. 可用性
- Vercel + Supabase の無料枠で運用開始
- 月間10万PVまでは無料枠で対応可能な設計

### 3-4. ユーザビリティ
- スマートフォン・PC両対応（レスポンシブ）
- 診断所要時間：5〜7分
- 8問を一画面に表示し、スクロールで回答できる

---

## 4. 制約事項

- 結果URLを失っても、ログイン済みであればマイページから再アクセス可能
- メール登録は任意だがアカウント機能を使う場合は必須

---

## 5. 用語定義

| 用語 | 定義 |
|------|------|
| 結果トークン | 自己診断完了時に発行される結果ページ専用URL識別子 |
| 友人トークン | 他者診断用URLの識別子（結果トークンとは別） |
| 自己像 | 自己診断の回答から算出したタイプ |
| 他者像 | 友人診断の回答平均から算出したタイプ |
| ギャップ | 自己像と他者像の差分 |
| 左極 | 各軸の左側（F/Y/K/T：即断・社交・感情発露・共感重視） |
| 右極 | 各軸の右側（G/R/H/S：熟考・内向・冷静・論理重視） |
