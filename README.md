# ミラー診断

> 「自分が思う自分」と「他人が見る自分」のギャップを可視化する、日本人向け対人関係特化型性格診断

**本番URL：** https://mirror-shindan.vercel.app
**GitHub：** https://github.com/KokiMoriguchi0812/mirror-shindan

## 概要

MBTIにヒントを得た日本人向けオリジナル性格診断。
自己診断と他者診断（友人・パートナーからの評価）を比較し、自己像と他者像のギャップを可視化する。

---

## ジョハリの窓 ── このアプリで何がわかるか

心理学の概念「ジョハリの窓」は、自己認識を4つの領域に分類します。

```
              自分が知っている   自分が知らない
            ┌─────────────────┬─────────────────┐
他人が       │   開放の窓       │   盲点の窓       │
知っている   │ (自他ともに知る) │ (他人だけ知る)   │
            ├─────────────────┼─────────────────┤
他人が       │   秘密の窓       │   未知の窓       │
知らない     │ (自分だけ知る)   │ (誰も知らない)   │
            └─────────────────┴─────────────────┘
```

**ミラー診断でわかること：**

- **自己診断** → 自分が思う自分（開放の窓 ＋ 秘密の窓）
- **友人診断** → 他人から見た自分（開放の窓 ＋ 盲点の窓）
- **ギャップ比較** → **「盲点の窓」を照らす**　＝　自分では気づかない自分の特性を発見できる

---

## 特徴

- **友人URLシェア**：診断後に専用URLを友人に送るだけで他者診断が完了
- **複数人平均**：複数の友人の回答を平均して「他者像」を算出
- **16タイプ**：4軸の組み合わせで16種類のキャラクタータイプを判定
- **アカウント登録任意**：ゲストでも診断・結果確認が可能

## 診断の4軸

| 軸 | 一方 | もう一方 |
|----|------|---------|
| 行動様式 | 即断即行（風） | 熟考慎重（岩） |
| 対人距離 | 社交開放（野） | 内向静穏（林） |
| 感情表現 | 感情発露（火） | 冷静沈着（氷） |
| 価値基準 | 共感重視（月） | 論理重視（山） |

## 環境構築手順

```bash
# リポジトリをクローン
git clone https://github.com/KokiMoriguchi0812/mirror-shindan.git
cd mirror-shindan

# 依存関係インストール
npm install

# 環境変数設定
cp .env.example .env.local
# .env.local に以下を設定（Supabaseダッシュボードから取得）

# 開発サーバー起動
npm run dev
```

## 環境変数

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```
